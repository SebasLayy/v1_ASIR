## [*] ¿Qué es esto?

**v1_ASIR** es mi repositorio donde guardo todo el trabajo de clase de mi primer año del ciclo superior de **Administración de Sistemas Informáticos en Red (ASIR)**. Aquí están apuntes, ejercicios, proyectos y pequeños experimentos.

Esto sirve para:
- [#] Mi memoria personal para no olvidar nada de cara al segundo año.
- [#] Una guía para futuros estudiantes de ASIR que quieran ver ejemplos prácticos.
- [#] Un laboratorio para practicar prompts de IA aplicados a HTML, CSS, lenguajes de marcas y Java (y Python).
- [#] Un lugar con varias webs (aunque no subidas directamente, hay que descargar el repo y abrir los archivos .html).
- [#] Un baúl de recursos para repasar redes, bases de datos, ISO, programación y más.

## [*] Asignaturas que abarca

[REDES] Configuración, subneteo, cables, switches, routers, Wireshark…

[BASES DE DATOS] Modelo entidad-relación, modelo relacional, consultas con Oracle Database, INSERT, actualizaciones, etc.

[ISO] Implantación de Sistemas Operativos: Linux, Windows, virtualización, scripting.

[LENGUAJE DE MARCAS] XML, JSON, HTML5, CSS3, XSLT, y webs varias.

[PROGRAMACION] Python (bucles, orientación a objetos, fundamentos). También algo de Java para aparentar.

Además hay una sección donde uso prompts de inteligencia artificial para generar fragmentos de código, explicaciones y ejercicios.

1. Recordar todo en segundo curso – No olvidar cómo se configura una red o se hace una consulta en Oracle.
